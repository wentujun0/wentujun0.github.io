Java中的基本数据类型

java中的基本数据类型与其对应的引用数据类型

基本数据类型 名称 引用数据类型(包装类)
int 整型 Integer
short 短整型 Short
long 长整型 Long
byte 位 Byte
float 单精度浮点型 Float
double 双精度浮点型 Double
boolean 布尔类型 Boolean
char 字符类型 Character

1、Integer、Byte、Float、Double、Short、Long都属于Number类的子类，Number类本身提供了一系列的返回以上6中基本类型的操作。

2、Character属于Object的直接子类

3、Boolean属于Object的直接子类。

装箱与拆箱

装箱：将一个基本数据类型变为包装类。

拆箱：将一个包装类变为基本数据类型。

JDK1.5之后提供了自动的装箱与拆箱操作

Java中数据类型的转换

1、自动转换

byte<short<int<long<float<double<string

小转大（都可以向String类型转换）

2、强制类型转换

可能会有类型丢失

3、特殊的类型转换

char或者String 转换成int 或者float

Integer.parseInt(str1);

Float.parseFloat(str2);
