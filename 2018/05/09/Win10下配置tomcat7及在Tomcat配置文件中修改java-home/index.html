Win10下配置tomcat7及在Tomcat配置文件中修改java_home

发表于 2018-05-09 |

在WIn10系统中配置Tomcat环境变量

在配置Tomcat环境变量前先配置jdk，确保jdk无误后再配置Tomcat。

注意：Tomcat以及JDK都需要安装在纯英文目录下！！

Tomcat安装目录：

方法一、在“此电脑”上配置环境变量（常有错误）

环境变量：

系统变量上点击”新建“,在弹出的新建系统变量提示框：

“变量名”栏填写：CATALINA_HOME（要大写，HOME前有下划线）

“变量值”栏填写：D:\software\apache-tomcat-7.0.84

点击确定。

配置Path变量

选中Path变量，点击右下角“编辑”。弹出框中选择“新建”，在光标闪烁的空白处填写：

D:\software\apache-tomcat-7.0.84

即：文件路径名+bin，一路“确定”退出环境变量配置。 配置完成

方法二、在Tomcat配置文件上配置环境变量

用方法一配置环境变量，常常出错，而且还会有错误”JRE_HOME”

The JRE_HOME environment variable is not defined correctl

This environment variable is needed to run this program

配置方法:

1、tomcat/bin/setclasspath.bat 用记事本打开

2、添加添加两行代码java根目录和Jre根目录

set JAVA_HOME=D:\software\jdk1.7.0_80

set JRE_HOME=D:\software\jdk1.7.0_80
