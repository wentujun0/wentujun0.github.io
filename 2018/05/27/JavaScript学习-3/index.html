面向对象

基本数据类型包装类

为了便于操作基本类型值，ECMAScript 提供了3个特殊的引用类型：Boolean,Number,String。它们是引用类型。当读取基本数据类型时，后台就会创建一个对应的基本包装类对象，所以我们在操作基本数据类型时，可以直接调用一些方法。

序号 方法 作用
1、 anchor() 生产锚点
2、 blink() 为元素添加blink标签
3、 charAt() 返回指定索引位置处的字符
4、 charCodeAt() 回一个整数，代表指定位置上字符的 Unicode 编码
5、 fontcolor() 把带有 COLOR 属性的一个 HTML 标记放置在 String 对象中的文本两端
6、 indexOf() 返回 String 对象内第一次出现子字符串的字符位置
7、 italics() 把 HTML 标记放置在 String 对象中的文本两端
8、 link() 把一个有 HREF 属性的 HTML 锚点放置在 String 对象中的文本两端
9、 replace() 返回根据正则表达式进行文字替换后的字符串的复制
10、 split() 切割
11、 Substr() 截取子串
12、 toUpperCase() 转大写
13、 toLowerCase 转小写
